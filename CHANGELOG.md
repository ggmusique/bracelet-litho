# Historique des versions

## v1.0.0 — 2026-05-31

- Première version stable
- Gestion des pierres, bracelets et produits de lithothérapie
- Onglets : Pierres / Bracelets / Produits / Étiquettes / Impression
- Grille d'impression Action A4 — 3 colonnes × 8 lignes (24 étiquettes)
- Étiquettes 70 × 37 mm, zéro espace entre elles, marge d'origine 2 mm
- Éditeur Zoom pour positionner les éléments sur l'étiquette
- Profils de mise en page JSON (Bracelet / Vertus)
- Affichage configurable des prix : PV / PR / Marge (activables indépendamment)
- Calcul automatique du prix de revient et de la marge
- Export PDF des étiquettes et des planches A4
- Impression directe via win32print
- Sauvegarde / restauration des données JSON
- Mise à jour automatique depuis GitHub
- Système de versionnage (version.py + CHANGELOG.md)
